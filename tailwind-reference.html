Tailwind CSS Reference

よく使うTailwind class早見表

このページは制作中にclassをすぐ確認するためのローカル参照用です。実装では必要なclassだけを各EJSに移してください。

Responsive

Tailwindはmobile firstです。通常classがSP基準、`md:`や`lg:`を付けたclassが指定幅以上で上書きされます。

block md:flex

SPではblock、768px以上ではflexに変更。

grid grid-cols-2 md:grid-cols-4

SPは2カラム、768px以上では4カラム。

px-4 md:px-8 lg:px-10

左右余白を画面幅に応じて段階的に広げる。

text-sm md:text-base lg:text-lg

文字サイズをSP、タブレット、PCで切り替える。

Layout

要素の並び、中央寄せ、グリッド、絶対配置など、レイアウトの骨格を作るclassです。

Container

mx-auto max-w-4xl px-4

`mx-auto`で中央寄せ、`max-w-*`で最大幅を制限します。

Flex

items-center justify-between

横並びの配置に使います。ヘッダーやボタン列に向いています。

Grid

1 2 3 4

商品一覧など、カラム数を決めて整列させる時に使います。

Position

absolute right-3 top-3

親に`relative`、子に`absolute`を付けて位置を固定します。

Spacing

`p`はpadding、`m`はmarginです。`t/r/b/l`は方向、`x/y`は左右・上下を表します。例: `mt-4`はmargin-top、`px-4`は左右padding。

p-2
p-4
p-8
mt-4 mb-8

上余白と下余白を個別指定。

px-4 py-6

左右と上下の内側余白を指定。

gap-4 md:gap-6

grid/flex内の要素間隔を指定。

space-y-4

縦に並ぶ兄弟要素の間隔を指定。

Typography

文字サイズ、太さ、行間、フォントファミリーを指定します。見出しは`font-bold`と`leading-tight`、本文は`leading-7`などを使うと読みやすいです。

text-xs text-slate-500

補足や小さいラベル向け。

text-sm leading-6 text-slate-600

小さめ本文。`leading-6`で行間を確保。

text-base font-bold leading-7

通常サイズの強調テキスト。

text-2xl md:text-4xl font-bold leading-tight

レスポンシブな見出し。

font-serif text-xl

明朝系フォント。和風・高級感の表現向け。

font-roboto text-xl font-bold

英数字や価格表示向け。

Color

`bg-*`は背景色、`text-*`は文字色、`border-*`は線色です。`/90`のような指定は透明度を表します。

bg-black text-white
bg-pickup-bg
bg-category-loop
bg-slate-900
text-black/90

黒の90%不透明。本文向け。

text-link-muted

設定済みの薄いリンク色。

border-slate-200

薄いグレーの境界線。

bg-white/80

白背景の80%不透明。

Size / Image

画像は固定pxより`aspect-*`と`object-cover`を優先します。正方形の商品画像は`aspect-square`が基本です。

aspect-square object-cover

aspect-video object-cover

h-32 w-32 rounded-full

State

`hover:`、`focus:`、`disabled:`のように状態名をprefixにすると、その状態の時だけstyleが効きます。

hover / focus

Common Snippets

この案件でよく使う組み合わせです。セクションや商品カードを作る時の初期値として使えます。

class="mx-auto max-w-pickup px-sp-section-x md:px-8"

中央寄せコンテナ。SPは可変padding、PCは固定padding。

class="grid grid-cols-2 gap-sp-gap-md md:grid-cols-4 md:gap-6"

商品一覧向け。SP 2カラム、PC 4カラム。

class="aspect-square overflow-hidden bg-white"

商品画像枠。正方形を維持してはみ出しを隠す。

class="transition duration-300 ease-out group-hover:scale-subtle"

親hover時に控えめな拡大アニメーション。

class="sr-only"

画面には出さず、スクリーンリーダーには読ませるテキスト。
